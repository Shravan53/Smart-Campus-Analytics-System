# Smart Campus Analytics System

This project analyzes student activity and campus resource usage using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Overview
The dataset includes student attendance, study hours, lab usage, and energy consumption.

Data is preprocessed and analyzed to:
- Predict student productivity
- Analyze energy efficiency
- Detect anomalies in campus data

Machine learning models like Decision Tree and Isolation Forest are used.

## Features
- Productivity Prediction (High / Medium / Low)
- Energy Efficiency Analysis
- Anomaly Detection
- Insights using Data Visualization
